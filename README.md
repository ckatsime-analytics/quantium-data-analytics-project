# Quantium Data Analytics Project 

## Overview
This project was completed as part of the Quantium Data Analytics Virtual Internship.  
It analyzes retail transaction and customer data to uncover sales trends, product performance, and customer behavior, with the goal of generating actionable business insights.

## Objectives
1. Load transaction dataset and validate structure  
2. Perform exploratory data analysis (EDA)  
3. Convert date column to datetime format  
4. Analyze daily and quarterly sales trends  
5. Compare yearly sales (2018 vs 2019)  
6. Aggregate sales by product  
7. Load customer dataset  
8. Merge transaction and customer data  
9. Detect outliers using IQR  
10. Segment customers by lifestage and premium status  
11. Create pivot tables for combined segmentation  
12. Calculate percentage contribution of each segment  

## Key Insights
- **Sales Trends**  
- Sales declined ~2% from 2018 (₹977k) to 2019 (₹957k).  
- Quarterly regression shows a slight downward slope, signaling tapering demand.  

- **Product Performance**  
- **Top sellers**: Dorito Corn Chip Supreme 380g, Smiths Crinkle Chips (Salt & Vinegar, Original), Kettle Mozzarella Basil & Pesto 175g, Cheezels Cheese 330g.  
- **Weak performers**: Woolworths Medium/Mild Salsa 300g, WW Crinkle Cut Original 175g, Sunbites Wholegrain Crisps (various flavors).  
- **Customer Segmentation (Lifestage)**  
  - Strongest contributors: Older Singles/Couples (₹402k), Retirees (₹366k), Older Families (₹353k).  
  - Lowest: New Families (₹50k).  

- **Customer Segmentation (Premium Status)**  
- Mainstream customers dominate with ₹750k sales.  
- Budget customers contribute ₹676k.  
- Premium customers lag at ₹507k.  

- **Combined Segmentation (Lifestage × Premium)**  
  - Young Singles/Couples: 60% sales from mainstream.  
  - Older Families: 48% sales from budget.  
  - Retirees: Balanced mix, mainstream slightly ahead.  
  - Older Singles/Couples: Equal distribution across all three segments (~33% each).  

## Recommendations
- **Product Strategy**  
  - Expand marketing and shelf space for top-selling chips.  
  - Reassess Woolworths-branded salsa and Sunbites — consider reformulation, rebranding, or discontinuation.  

- **Customer Targeting**  
  - Focus promotions on Older Singles/Couples and Retirees.  
  - Strengthen loyalty programs for mainstream and budget customers.  
  - Explore premium product bundles for Older Families to increase margins.  

- **Sales Growth Strategy**  
  - Counter declining sales with seasonal campaigns (family packs, festive bundles).  
  - Introduce cross-promotions (chips + dips) to boost underperforming salsa products.  
  - Use targeted advertising by lifestage (health-conscious messaging for retirees, convenience for young families).
